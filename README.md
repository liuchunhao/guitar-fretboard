# Guitar Fretboard Visualizer

一個用於記憶吉他音階指型的網站。

## Features

*   **Interactive Fretboard:** A visual representation of a 24-fret guitar fretboard with permanent labels for frets (X-axis) and strings (Y-axis).
*   **Note Display on Hover:** Hover your mouse over any fret to see the name of the note, the string number, and the fret number.
*   **Scale Highlighting:** Select a scale from the dropdown menu to highlight all its notes on the fretboard. Available scales include C Major, G Major, A Minor, and F Lydian.

## How to Run

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```
